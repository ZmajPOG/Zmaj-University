Zmaj University - Faculty web application

A full-stack faculty information system for managing users, courses, exam periods and student records.

Three-role architecture (admin / professor / student) with separated responsibilities and dashboards
Course management, enrollments, exam periods, student exam applications and grading
PHP + PDO + MySQL backend, secure session handling, prepared statements (no SQL concatenation)
Clean responsive UI with Bootstrap + custom theme, role-specific navigation and stats
   
Technologies: PHP, PDO/MySQL, Bootstrap, HTML/CSS, JS (vanilla), model-viewer (GLB), optional assets (Figma + ODT/PDF docs).
